# NJTools - High-Performance Web Utilities

NJTools is a lightweight, high-performance suite of web utility tools designed for developers and power users. Built with **React 19**, **TypeScript**, and **Tailwind CSS**, it focuses on speed, privacy, and clean UI/UX. All processing is done client-side, ensuring user data never leaves the browser.

[![Site Status](https://img.shields.io/website?url=https%3A%2F%2Fnjtools.xyz)](https://njtools.xyz)
[![License](https://img.shields.io/badge/license-All%20Rights%20Reserved-red)](./LICENSE)

---

## 🛠 Features

### 1. Media Tools
* **SVG to PNG Converter:** Convert vector graphics to high-resolution raster images instantly.
* **PNG to SVG (Tracing):** Advanced image tracing using `imagetracerjs` to convert raster images back to scalable vectors.

### 2. Developer Utilities
* **JSON Formatter:** Prettify, minify, and validate JSON structures with syntax highlighting.
* **Unit Converter:** Precision conversion for Length and Weight with instant feedback.

### 3. Core Architecture
* **Clean Architecture:** Modular component design for easy scalability.
* **SEO Optimized:** Dynamic meta tags and SSR-ready structure using `react-helmet-async`.
* **Privacy First:** Zero server-side data storage; 100% client-side execution.

---

## 🚀 Tech Stack

* **Frontend:** React 19 (Functional Components, Hooks)
* **Styling:** Tailwind CSS (Modern Utility-First UI)
* **Language:** TypeScript (Strict Typing for Robustness)
* **Icons:** Lucide-React
* **Analytics:** Google Analytics 4 (GA4) integration via `react-ga4`

## 📦 Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/niduranga/NJTools.git](https://github.com/niduranga/NJTools.git)
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run Development Server:**
    ```bash
    npm run dev
    ```

4.  **Build for Production:**
    ```bash
    npm run build
    ```
